# Diet Management System

このディレクトリはhikariのダイエット管理用プロジェクトです。

## 構成

```
diet/
├── CLAUDE.md              # このファイル
├── goals.md               # 目標設定
├── checklist_master.md    # チェックリスト項目マスター（編集用）
├── data/                  # データ
│   ├── checklist/         # チェックリスト回答
│   │   ├── YYYY-MM-DD.csv # 回答データ
│   │   └── YYYY-MM-DD.md  # 結果サマリー
│   ├── weight.csv         # 体重記録
│   ├── meals.csv          # 食事記録
│   └── exercise.csv       # 運動記録
├── recipes/               # レシピ管理
│   ├── ingredients.csv    # 材料マスター
│   └── items/             # 個別レシピ（MDファイル）
│       └── レシピ名.md
└── logs/                  # 日々の記録（Markdown）
    └── YYYY-MM-DD.md      # 日別ログ
```

## 参照ファイル

> チェックリスト関連の操作時は必ず参照すること

- **[checklist_master.md](./checklist_master.md)** - チェック項目のマスターデータ
  - 項目の追加・変更・削除はここで行う
  - Lv1/Lv2/Lv3 の項目定義とスコア基準を含む

## Claudeへの指示

### 記録の追加

ユーザーが体重、食事、運動を報告したら：

1. **CSVファイルに追記**
   - 体重 → `data/weight.csv`
   - 食事 → `data/meals.csv`
   - 運動 → `data/exercise.csv`

2. **日別ログに記録**
   - `logs/YYYY-MM-DD.md` に追記（なければ作成）

### 分析・レポート

- 「今週の振り返り」→ 直近7日間のデータを集計してサマリー表示
- 「進捗確認」→ 目標に対する現在の達成状況を表示
- 「グラフ」→ CSVデータをもとにASCIIグラフやトレンドを表示

### 入力例

```
今日の体重：72.5kg
朝食：トースト、卵、コーヒー（推定400kcal）
昼食：サラダチキン弁当（500kcal）
夕食：刺身定食（600kcal）
運動：ウォーキング30分
```

このように自然に話しかけるだけでOK。Claudeが適切にパースして記録します。

### コマンド一覧

**記録系**
- `記録して` - 今日の記録を追加
- `今日のまとめ` - 今日の摂取カロリー・消費カロリーを表示
- `今週の振り返り` - 週間サマリー
- `目標を設定` - 新しい目標を設定
- `進捗確認` - 目標に対する進捗を確認

**チェックリスト系**
- `チェックリスト` - 習慣チェックを実施（いつでも実行可）
- `項目を追加` - チェックリストに新しい項目を追加（checklist_master.mdを編集）
- `項目を編集` - チェックリストの項目を変更（checklist_master.mdを編集）

### レシピ管理

**データファイル**
- `recipes/ingredients.csv` - 材料マスター（名前、カテゴリ、単位あたり栄養素）
- `recipes/items/*.md` - 個別レシピファイル（1レシピ1ファイル）

**ingredients.csv カラム**
`id, name, category, unit, kcal_per_unit, protein_per_unit, fat_per_unit, carb_per_unit, note`
- category: 肉類/魚介類/野菜/果物/穀類/乳製品/調味料/その他
- unit: g, ml, 個, 枚 など

**レシピMDファイルのテンプレート** (`recipes/items/レシピ名.md`)
```markdown
# レシピ名

- カテゴリ: 主菜/副菜/汁物/主食/おやつ/ドリンク
- 分量: ○人前
- タグ: 簡単, 高タンパク, 作り置き

## 栄養素（1人前）

| kcal | タンパク質 | 脂質 | 炭水化物 |
|------|-----------|------|---------|
| 000  | 00g       | 00g  | 00g     |

## 材料

| 材料 | 分量 |
|------|------|
| xxx  | 000g |

## 作り方

1. ...
2. ...

## メモ

（任意）
```

**コマンド**
- `材料を追加` - 材料マスターに新しい材料を追加
- `レシピを追加` - 新しいレシピをMDファイルとして登録
- `レシピ検索` - タグ・カテゴリ・カロリーでレシピを検索
- `献立提案` - 目標カロリーに合わせてレシピから献立を提案
- `レシピ一覧` - 登録済みレシピの一覧表示

**食事記録との連携**
- 食事記録時にレシピ名が指定された場合、該当MDファイルからカロリー・PFCを自動取得して meals.csv に記録

### チェックリスト運用ルール

1. **項目の変更**はすべて `checklist_master.md` で行う
2. チェック実施時は `checklist_master.md` を読み込んで最新の項目を使う
3. 結果は `data/checklist/` に日付ごとに保存
   - `YYYY-MM-DD.csv` - 回答データ
   - `YYYY-MM-DD.md` - 結果サマリー
4. スコア基準も `checklist_master.md` に定義されている
