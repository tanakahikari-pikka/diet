# 鳥肉ときのこの炊き込みご飯

- カテゴリ: 主食
- 分量: 4人前
- タグ: 作り置き, 高タンパク

## 栄養素（1人前）

| kcal | タンパク質 | 脂質 | 炭水化物 |
|------|-----------|------|---------|
| 533 | 36.2g | 9.5g | 72.0g |

※栄養素は推定値です（鶏皮・脂身除去）

## 材料

| 材料 | 分量 |
|------|------|
| 鳥もも肉（皮・脂身除去） | 600g |
| えのき | 1パック |
| まいたけ | 1パック |
| 椎茸 | 1パック |
| 米 | 2合 |
| 醤油 | 大さじ3 |
| みりん | 大さじ3 |
| 料理酒 | 大さじ3 |
| 砂糖 | 大さじ1 |

## 作り方

1. きのこを1cmカット
2. きのこに塩を振って乾煎りする
3. 水分が飛んで、焦げ目がついたら
4. 一口小にカットした鶏肉に塩を振って投入
5. 焼き目がつくまで待って、ひっくり返した後は弱火
6. 料理酒とみりんを投入、アルコールを飛ばす
7. 醤油と砂糖を投入、軽く絡める
8. 具と炊き上がった米を混ぜる
9. 取り分けて冷凍
