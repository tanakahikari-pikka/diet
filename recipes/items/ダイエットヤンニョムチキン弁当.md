# ダイエットヤンニョムチキン弁当

- カテゴリ: 弁当
- 分量: 4人前
- タグ: 高タンパク

## 栄養素（1人前）

| kcal | タンパク質 | 脂質 | 炭水化物 |
|------|-----------|------|---------|
| 551  | 35g       | 11g  | 74g     |

## 材料

### ご飯

| 材料 | 分量 |
|------|------|
| お米 | 2合 |
| エノキ | 1株 |
| 塩 | ふたつまみ |
| 味の素 | 5振り |
| 料理酒 | 大さじ1 |
| 水 | 340ml |

### ヤンニョムソース

| 材料 | 分量 |
|------|------|
| コチュジャン | 大さじ2 |
| ケチャップ | 大さじ1 |
| 蜂蜜（砂糖、甘味料でも） | 大さじ1 |
| みりん | 大さじ1 |
| お酢 | 大さじ1/2 |
| 五香粉（シナモン） | 小さじ1/3 |

### チキン

| 材料 | 分量 |
|------|------|
| 鶏もも肉 | 600g |
| 料理酒 | 大さじ2 |
| 塩胡椒 | 7振り |
| おろしニンニク | 6cm |
| 薄力粉 | 大さじ1 |
| 片栗粉 | 大さじ1 |
| ごま油 | 大さじ1/2 |

## 作り方

1. 炊飯釜に洗ったお米とご飯の調味料（塩、味の素、料理酒、水）を入れ、エノキを手でほぐして加え混ぜて炊飯する
2. ヤンニョムソースの調味料（コチュジャン、ケチャップ、蜂蜜、みりん、お酢、五香粉）を混ぜておく。鶏肉の皮と余分な脂を取り除き一口大に切る
3. ボウルに切った鶏肉、料理酒、おろしニンニクを入れ揉み込み、薄力粉と片栗粉を加えて軽く馴染ませる
4. 冷たいフライパンにごま油と鶏肉を入れ中火で焼き色をつけ、裏返して弱火で焼く。混ぜたタレを回しかけて絡める
5. 保存容器にご飯とヤンニョムチキンを分けて盛り付け、粗熱を取り蓋をして冷凍保存する
6. 食べる際は蓋をずらし600Wのレンジで約5分加熱する
