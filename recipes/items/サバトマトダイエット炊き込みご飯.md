# サバトマトダイエット炊き込みご飯

- カテゴリ: 弁当
- 分量: 4人前
- タグ: 作り置き, 高タンパク, 簡単
- 出典: [だれウマ](https://www.youtube.com/watch?v=iJWh1_knI34)

## 栄養素（1人前）

| kcal | タンパク質 | 脂質 | 炭水化物 |
|------|-----------|------|---------|
| 382 | 13.0g | 7.0g | 65.0g |

※栄養素は推定値です

## 材料

| 材料 | 分量 |
|------|------|
| 米（研いでおく） | 2合 |
| 鯖の水煮缶 | 1缶 |
| めんつゆ（2倍タイプ） | 大さじ3 |
| 料理酒 | 大さじ2 |
| 水 | 1.5合と2合の線の間まで |
| トマト（ヘタは取る） | 1個 |
| まいたけ（手でほぐす） | 1パック |
| 生姜（千切り） | 10g |
| 塩昆布 | 5g |
| 大葉（千切り） | 5〜10枚 |

## 作り方

1. 炊飯釜に洗った米、鯖の水煮缶の汁、めんつゆ、料理酒を入れ、水を1.5合と2合の線の間まで加える。トマト、まいたけ、生姜、塩昆布を加えて炊飯する

> **ポイント:** 水は2合の線まで入れるとトマトや鯖缶の水分でベチャつくので、必ず1.5合と2合の間まで

> **ポイント:** 料理酒で鯖の生臭みを抑え、ご飯もふっくら艶やかに。生姜もさらに臭み取り＆さっぱり効果

> **ポイント:** トマトのグルタミン酸 × 鯖缶のイノシン酸 × きのこのグアニル酸 → うま味の相乗効果

> **痩せポイント:** 鯖缶のオメガ3脂肪酸（DHA・EPA）は中性脂肪を減らし代謝アップ。白米と一緒に摂ると血糖値の急上昇も抑制

> **痩せポイント:** トマトのリコピンで代謝アップ・脂肪燃焼促進。まいたけのMXフラクションで中性脂肪・悪玉コレステロールの分解促進

2. 炊飯後、大葉を加えよく混ぜ合わせたら完成。男性は4食分、女性は5食分

> **冷凍保存:** 熱いうちにラップで包み、粗熱を取ってから冷凍庫へ。解凍は600Wのレンジで2分半
