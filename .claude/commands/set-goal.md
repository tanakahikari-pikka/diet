# ダイエット目標設定

対話形式でダイエット目標を設定し、goals.md を更新します。

## 手順

1. **現在の goals.md を読み込む**
   - `goals.md` を Read ツールで読み込み、現在の設定を確認

2. **AskUserQuestion で以下を順番に質問**

   質問1: 基本情報
   - 現在の体重（kg）
   - 目標体重（kg）
   - 目標期限（YYYY-MM-DD または「3ヶ月後」など相対指定も可）

   質問2: 1日の目標
   - 1日の摂取カロリー上限（kcal）- 未定なら空欄可
   - タンパク質目標（g）- 未定なら空欄可
   - 運動消費カロリー目標（kcal）- 未定なら空欄可

3. **goals.md を更新**
   - 回答をもとに goals.md を Edit ツールで更新
   - 開始日は今日の日付を自動設定

4. **設定内容のサマリーを表示**
   - 設定した目標を見やすく表示
   - 目標達成までの減量幅（kg）を計算
   - 1週間あたりの目安減量ペースを算出（期限がある場合）

## 出力フォーマット例

```
## 目標を設定しました

| 項目 | 値 |
|------|-----|
| 開始日 | 2025-02-15 |
| 開始体重 | 75.0 kg |
| 目標体重 | 68.0 kg |
| 目標期限 | 2025-05-15 |
| 減量目標 | -7.0 kg |
| 週あたり目安 | -0.54 kg/週 |

頑張りましょう！
```

## 注意事項

- 週 0.5〜1.0kg 以上の減量ペースになる場合は、健康的なペースについてアドバイスを添える
- 未入力の項目は「未設定」として保持し、後から追加できるようにする
