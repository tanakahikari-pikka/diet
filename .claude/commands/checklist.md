# 習慣チェックリスト回答

対話形式でチェックリストに回答し、結果を記録します。

## 参照ファイル

$FILE:checklist_master.md

## 手順

1. **チェックリストマスターを確認**
   - 上記で読み込まれた `checklist_master.md` の項目を使用

2. **チェックリストの種類を選択**
   - AskUserQuestion で「初回診断」か「週間チェック」かを質問

3. **レベルごとに質問**

   ### 初回診断の場合
   各レベル（Lv1→Lv2→Lv3）ごとに AskUserQuestion で質問:
   - multiSelect: true で複数選択可能に
   - 「できている項目をすべて選んでください」形式

   ### 週間チェックの場合
   各レベル（Lv1→Lv2→Lv3）ごとに AskUserQuestion で質問:
   - multiSelect: true で複数選択可能に
   - 「今週できた項目をすべて選んでください」形式

4. **スコアを集計**
   - 各レベルのスコアと達成率を計算
   - 合計スコアを算出

5. **結果ファイルを更新**
   - 初回診断 → `checklist_initial.md` を更新
   - 週間チェック → `checklist_weekly.md` を更新
   - チェックした項目に [x] をつける
   - スコアを記入

6. **結果サマリーを表示**

## 出力フォーマット例

```
## チェックリスト結果

**種類:** 週間チェック
**期間:** 2月9日〜2月15日

| レベル | スコア | 達成率 |
|--------|--------|--------|
| Lv1（基礎習慣） | 5 / 6 | 83% |
| Lv2（成長習慣） | 4 / 6 | 67% |
| Lv3（理想習慣） | 2 / 5 | 40% |
| **合計** | **11 / 17** | **65%** |

### 診断
基礎はできつつある。Lv1完璧を目指そう

### 今週の課題
- [ ] 21時以降の食事を控える
```

## 注意事項

- Lv1 で達成できていない項目があれば、優先課題として提示
- 前回からの変化があれば言及（週間チェックの場合）
