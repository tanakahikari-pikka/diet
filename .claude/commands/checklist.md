# 習慣チェックリスト回答

対話形式でチェックリストに回答し、結果を記録します。いつでも実行可能。

## 参照ファイル

- `checklist_master.md` - 項目マスター（必ず最初に読み込む）
- `data/checklist/` - 過去の回答データ（比較用）

## 手順

### 1. チェックリストマスターを読み込む

`checklist_master.md` を読み込んで最新の項目を取得。

### 2. 過去データを確認（任意）

`data/checklist/` 内の最新ファイルがあれば読み込み、前回との比較に使用。

### 3. レベルごとに質問

各レベル（Lv1→Lv2→Lv3）ごとに AskUserQuestion で質問:
- multiSelect: true で複数選択可能に
- 「現在できている項目をすべて選んでください」形式
- 項目数が多い場合は分割して質問（4項目ずつ）

### 4. 結果を保存

#### CSV（1次データ）: `data/checklist/YYYY-MM-DD.csv`

```csv
item_id,item,level,category,score,achieved
1,1日3食食べている,Lv1,食事,2,1
2,毎食タンパク質を摂っている,Lv1,食事,3,0
...
```

| カラム | 説明 |
|--------|------|
| item_id | 項目ID |
| item | 項目名 |
| level | レベル（Lv1/Lv2/Lv3） |
| category | カテゴリ（食事/生活/運動） |
| score | 配点 |
| achieved | 達成（1=達成, 0=未達成） |

#### MD（分析）: `data/checklist/YYYY-MM-DD.md`

```markdown
# チェックリスト分析

**実施日:** YYYY年M月D日
**データ:** [YYYY-MM-DD.csv](./YYYY-MM-DD.csv)

---

## スコアサマリー

| レベル | スコア | 達成率 |
|--------|--------|--------|
| Lv1（基礎習慣） | X / 19 | XX% |
| Lv2（成長習慣） | X / 19 | XX% |
| Lv3（理想習慣） | X / 26 | XX% |
| **合計** | **X / 64** | **XX%** |

---

## 診断

**現状:** （スコアに基づいた現状分析）

**強み:**
- （達成している項目から抽出）

**弱み:**
- （未達成の項目から抽出）

---

## ネクストアクション

### 最優先（Lv1未達成・高配点）

| 項目 | 配点 | 具体的アクション |
|------|:----:|------------------|
| （項目名） | X | （具体的な行動提案） |

### 今週のフォーカス

**「（1つに絞った最優先タスク）」**

- （なぜこれが重要か）
- （具体的な実行方法）

---

## 次回チェック時の確認ポイント

- [ ] （確認項目1）
- [ ] （確認項目2）
```

### 5. 結果サマリーを表示

スコアサマリーとネクストアクションをユーザーに表示。

## 分析のポイント

### スコア基準（checklist_master.md参照）

| Lv1達成率 | 状態 |
|-----------|------|
| 50%未満 | まずLv1の基礎固めから |
| 50-79% | 基礎構築中。Lv1完璧を目指す |
| 80%以上 | Lv2に挑戦 |

### ネクストアクションの優先順位

1. **Lv1の未達成項目**を最優先（基礎ができていないと上位は意味がない）
2. **高配点（3点）の項目**を優先
3. **1つに絞る**（複数提示しても実行されない）

### 前回との比較（過去データがある場合）

- 改善した項目を称賛
- 後退した項目を警告
- 継続できている項目を強調

## 注意事項

- Lv1で達成できていない項目があれば、必ず優先課題として提示
- ネクストアクションは具体的な行動レベルで記載（「野菜を食べる」→「コンビニでサラダを1品追加する」）
- 1回のフォーカスは1項目に絞る
