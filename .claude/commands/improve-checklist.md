# チェックリスト改善提案

チェックリストの回答結果を分析し、項目の追加・修正を提案します。

## 手順

1. **現状データを読み込む**
   - `checklist_master.md` - 現在の項目定義
   - `checklist_initial.md` - 初回診断結果
   - `checklist_weekly.md` - 週間チェック結果
   - `goals.md` - ユーザーの目標

2. **分析を実施**

   ### パターン分析
   - 常にチェックされる項目 → 簡単すぎる可能性
   - 常にチェックされない項目 → 難しすぎる/不要な可能性
   - レベル間の達成率バランス

   ### ギャップ分析
   - goals.md の目標と現在の項目の整合性
   - 目標達成に必要だが項目にない習慣

3. **改善提案を作成**

   以下の観点で提案:
   - **追加すべき項目**: 目標に関連するが未カバーの習慣
   - **修正すべき項目**: 表現が曖昧、達成基準が不明確など
   - **削除/統合の候補**: 重複、形骸化している項目
   - **レベル変更の提案**: 難易度に応じたレベル移動

4. **提案を表示し、適用するか確認**
   - AskUserQuestion で各提案の採用/却下を確認
   - multiSelect で複数の提案を一度に選択可能

5. **採用された提案を適用**
   - `checklist_master.md` を Edit ツールで更新
   - 編集履歴に変更内容を追記

## 出力フォーマット例

```
## チェックリスト分析結果

### 現状サマリー
- 初回診断スコア: 12/19 (63%)
- 直近週間スコア: 11/17 (65%)

### 気づき
- Lv1 は安定して高得点（5-6/6）
- Lv3「週2回以上運動」が毎週未達成

---

## 改善提案

### 追加提案
1. **[Lv2] 週の運動計画を立てている**
   - 理由: 運動習慣が課題。計画があると実行しやすい

### 修正提案
2. **[Lv3] 週2回以上運動 → 週1回30分以上の運動**
   - 理由: ハードルが高すぎる可能性。段階的に

### レベル変更提案
3. **「間食をコントロール」を Lv2 → Lv1 へ**
   - 理由: 毎週達成できており、基礎習慣として定着

---

採用する提案を選んでください。
```

## 注意事項

- 一度に大きく変更しすぎない（1-3項目程度の変更を推奨）
- 変更理由を明確に説明する
- 編集履歴に日付と変更内容を記録する
