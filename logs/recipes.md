# 簡単常備菜レシピ

## 5分以内で作れる常備菜

### 1. 無限キャベツ
- キャベツをちぎってポリ袋に入れる
- ごま油 + 鶏ガラスープの素 + 塩を入れて揉む
- 冷蔵3日

### 2. もやしナムル
- もやしをレンジで2分チン
- 水気を切って、ごま油 + 醤油 + 鶏ガラスープの素
- 冷蔵2日

### 3. きゅうりの塩昆布和え
- きゅうりを叩いて一口大に
- 塩昆布 + ごま油で和える
- 冷蔵2日

### 4. ブロッコリーのおかか和え
- 冷凍ブロッコリーをレンジでチン
- めんつゆ + かつお節で和える
- 冷蔵3日

### 5. ピーマンの塩炒め
- ピーマンを細切りにしてフライパンで炒める
- 塩 + ごま油だけ
- 冷蔵4日

## タンパク質も摂れる系

### ツナキャベツ
- キャベツ + ツナ缶 + マヨ少々
- 冷蔵2日

### 卵とほうれん草炒め
- 冷凍ほうれん草 + 卵 + 醤油
- 冷蔵2日
